Tentang Saya

Saya adalah siswa dari jurusan Rekayasa Perangkat Lunak (RPL) di SMKN 4 Padalarang. Jurusan RPL berfokus pada pembelajaran tentang pengembangan perangkat lunak, pembuatan aplikasi, dan pemrograman untuk memenuhi kebutuhan dunia industri teknologi.

Di sini, kami belajar berbagai keterampilan teknologi modern, seperti:

  • Pengembangan aplikasi berbasis web, mobile, dan desktop
  • Pemrograman dengan bahasa seperti HTML, CSS, JavaScript, Python, PHP, dan Java
  • Pengelolaan basis data menggunakan MySQL dan MongoDB
  • Penguasaan tools seperti Git, Visual Studio Code, dan Figma

Kami percaya bahwa keterampilan ini tidak hanya mempersiapkan kami untuk masuk ke dunia kerja, tetapi juga mencetak generasi muda yang kreatif, inovatif, dan mampu menciptakan solusi teknologi yang bermanfaat.

Melalui website ini, kami ingin berbagi karya, proyek, dan pengalaman belajar kami. Anda bisa menemukan berbagai hasil kerja kami seperti:

  • Website interaktif
  • Aplikasi mobile sederhana
  • Desain antarmuka (UI/UX)
  • Proyek kolaborasi tim

Motto kami: "Belajar Koding, Bangun Masa Depan."

Terima kasih telah berkunjung ke halaman kami. Kami siap berinovasi dan menjadi bagian dari perkembangan teknologi masa depan!
